1st assessment test

The test will be held in a computer room, in a moodle environment quizz, in the moodle course [EIC0021-1S]  Métodos Numéricos. 
Some study material will be available online, namely formulas and course text.
The local computer resources will be fully available, for instance worksheets, CAS, programming IDE, calculators.
No other resources are allowed,  namely usb-pen, mobile, laptop, graphic or programmable calculators. Use of other comunications methods or the access to other moodle courses isn't allowed.
The test will be watched electronically by MoodleWatcher.

Students abide by the University regulations on ethics (in portuguese) Regulamento Disciplinar dos Estudantes da Universidade do Porto (despacho Reitoral GR.03/07/2011, CAP II, Artº 4º, Alínea l).

The test will focus on the following contents:


Theoretical knowledge:


  • Number representation;
  • Errors and error propagation;
  • Roots of real functions:
    • root isolation,
    • interval methods - bissection and false position,
    • initial guess,
    • non interval methods -Newton and Picard-Peano,
    • convergence,
    • closing /  stoping,
    • implementation eficiency.
  • Systems of non linear equations:  Newton and Picard-Peano;

Practical knowledge:
  • Number representation;
  • Errors and error propagation;
  • Roots of real functions:
    • root isolation,
    • interval methods - bissection and false position,
    • initial guess,
    • non interval methods -Newton and Picard-Peano,
    • convergence,
    • closing /  stoping,
    • implementation eficiency.
  • Systems of non linear equations:  Newton and Picard-Peano;
  • Quadrature and cubature:
    • Trapezoidal rule;
    • Simpson's rule;
    • Choice of integration step, the ratio of convergence and the error estimation;
    • Extension to the double integration.


Answer types

Students can answer:

  • calculating manual or using a spreadsheet, which should be delivered or submitted in the file submission application;
  • writing computer code in their choice programming language; Code, inputs and outputs should be uploaded using the file submissão application;  good programming can get a 15% bonus.
